# Pytorch-Autoencoder-

This model can use resample sound data.

Data file shape : .wav (10s)



System version
 - Python : 3.8+ 
 - conda : 4.9.1
 - Spyder : 4.1.5
 - Pytorch : 1.7.1
 - Numpy : 1.19.2
 - Librosa : 0.8.0
 - Pandas : 1.1.0
 
 
 model needs Librosa to upload .wav data
 
 How to install Librosa:
 
     Go to conda powershell prompt
 
     >>> pip install librosa
 
 
 when you write notebook, import librosa 
 
 
 
 Difficulties:
